# 🏛️ Sitio Web Oficial - Diputada Priscilla Castillo Gerli

Este proyecto es una maqueta (mockup) de alta fidelidad desarrollada como trabajo final para el **Bootcamp Frontend Sence**. El diseño está inspirado en la estética legislativa moderna de [Alexandria Ocasio-Cortez](https://ocasio-cortez.house.gov/), priorizando la rendición de cuentas, la transparencia y la cercanía con el distrito.

## 🎨 Concepto Visual
- **Paleta de Colores:** Uso audaz de Rojo Intenso y Rosado (AOC Style) para denotar energía y autoridad.
- **Tipografía:** Montserrat (títulos de alto impacto) e Inter (legibilidad moderna).
- **Enfoque:** Diseño orientado a la acción y visualización de proyectos de ley.

## 🛠️ Tecnologías Utilizadas
- **Framework:** [Vue 3](https://vuejs.org/) (Composition API)
- **UI Framework:** [Vuetify 3](https://vuetifyjs.com/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Iconos:** Material Design Icons (MDI)

## ✨ Características Destacadas
- **Publicador Dinámico:** Sistema reactivo que permite añadir noticias en tiempo real mediante un modal.
- **Sección Legislativa:** Tarjetas interactivas para mostrar los proyectos de ley en curso.
- **Feed de RRSS:** Diseño de cuadrícula (grid) para visualizar la actividad en terreno.
- **Formulario de Contacto:** Integración de UI para comunicación directa con los constituyentes del **Distrito 17**.
- **Responsive Design:** Totalmente adaptado para dispositivos móviles.
